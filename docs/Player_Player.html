Class: Player

Player#Player(maxHealth, maxEnergy, deck, hand, discard, effectopt)

new Player(maxHealth, maxEnergy, deck, hand, discard, effectopt)

Creates a player instance with health, energy, deck, cards, discard pile, and status effects and block.
Parameters:
Name Type Attributes Default Description
maxHealth number The player's maximum health.
maxEnergy number The player's maximum energy.
deck Deck The player's starting deck of cards.
hand Hand The player's starting cards of cards.
discard Discard The player's discard pile.
effect Array <optional>
 [] An array of status effects (e.g., poison, weaken). Defaults to an empty array.
